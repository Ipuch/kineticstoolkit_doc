kineticstoolkit.kinematics.track_rigid_body¶

kineticstoolkit.kinematics.track_rigid_body(kinematics, /, local_points, label='Trajectory', *, include_rigid_body=True, include_markers=False)¶

Track a rigid body from markers trajectories.

Warning

The function track_rigid_body is unstable, which means it may not be tested or settled yet. Please avoid using this function in production code.

Warning

The function track_rigid_body is deprecated since master and is scheduled to be removed in master. Use track_clusters().

This function tracks the specified rigid body in a TimeSeries that contains the required markers, and adds the tracked rigid body to a copy of the input TimeSeries as a Nx4x4 series of frames.

Parameters

  • kinematics (TimeSeries) – TimeSeries that contains at least the trajectories of the markers specified in rigid_body_definition.

  • local_points (Dict[str, ndarray]) – A dict where each key is a point name and its corresponding value is its local coordinates, as returned by ktk.kinematics.define_rigid_body().

  • label (str) – Name of the rigid body, that will be the data key in the output TimeSeries.

  • include_rigid_body (Optional.) – Include the tracked rigid body in the output TimeSeries as an Nx4x4 series. The default is True.

  • include_markers (Optional.) – Include the reconstructed markers in the output TimeSeries as a Nx4 series. Every marker of the rigid body definition is recontructed. The default is False.

Returns

A TimeSeries that contains the trajectory of the tracked rigid body.

Return type

TimeSeries

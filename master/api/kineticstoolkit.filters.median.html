kineticstoolkit.filters.median

kineticstoolkit.filters.median(ts, /, window_length=3)

Calculate a moving median.

On n-dimensional data, filtering occurs on the first axis (time).

Parameters

  • ts (TimeSeries) – Input TimeSeries

  • window_length (int) – Optional. Kernel size, must be odd. The default is 3.

Example

>>> ts = ktk.TimeSeries(time=np.arange(0, 0.5, 0.1))
>>> ts.data['data1'] = np.array([10., 11., 11., 20., 14., 15.])
>>> ts = ktk.filters.median(ts)
>>> ts.data['data1']
array([10., 11., 11., 14., 15., 15.])
Return type

TimeSeries

kineticstoolkit.kinematics.extend_cluster

kineticstoolkit.kinematics.extend_cluster(markers, /, cluster, new_point)

Add a point to an existing cluster.

Warning

The function extend_cluster is unstable, which means it may not be tested or settled yet. Please avoid using this function in production code.

Parameters

  • markers (TimeSeries) – TimeSeries that includes the new point trajectory, along with point trajectories from the cluster definition.

  • cluster (Dict[str, ndarray]) – The source cluster to add a new point to.

  • new_point (str) – The name of the point to add (data key of the markers TimeSeries).

Returns

The cluster with the added point.

Return type

Dict[str, np.ndarray]

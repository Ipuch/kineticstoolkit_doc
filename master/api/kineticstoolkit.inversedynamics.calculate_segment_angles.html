kineticstoolkit.inversedynamics.calculate_segment_angles

kineticstoolkit.inversedynamics.calculate_segment_angles(ts)

Calculate the segment’s projection angles in the three axes.

Warning

The function calculate_segment_angles is unstable, which means it may not be tested or settled yet. Please avoid using this function in production code.

Parameters

ts (TimeSeries) –

A TimeSeries with the following data keys:

  • ProximalJointPosition (Nx4)

  • DistalJointPosition (Nx4)

Returns

A copy of the input timeseries plus the ‘SegmentAngles’ data key, represented as an Nx3 numpy array.

Return type

TimeSeries

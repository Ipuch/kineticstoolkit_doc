kineticstoolkit.inversedynamics.calculate_com_position

kineticstoolkit.inversedynamics.calculate_com_position(ts, inertial_constants)

Calculate the position of the segment’s center of mass.

Warning

The function calculate_com_position is unstable, which means it may not be tested or settled yet. Please avoid using this function in production code.

Adds the data key ‘COMPosition’ to the TimeSeries based on the ‘COMProximalRatio’ key of the dict inertial_constants.

Parameters

  • ts (TimeSeries) –

    A TimeSeries with the following data keys:

    • ProximalJointPosition (Nx4)

    • DistalJointPosition (Nx4)

  • inertial_constants (Dict[str, float]) – A dict with at least a ‘COMProximalRatio’ key, which is the distance between the segment’s center of mass and the proximal joint, as a ratio of the distance between both joints.

Returns

A copy of the input timeseries plus the ‘COMPosition’ data key.

Return type

TimeSeries

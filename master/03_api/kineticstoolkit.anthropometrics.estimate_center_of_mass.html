kineticstoolkit.anthropometrics.estimate_center_of_mass

kineticstoolkit.anthropometrics.estimate_center_of_mass(markers, /, segments, *, sex='M')

Estimate the segments’ center of mass based on anthropometric data.

Warning

The function estimate_center_of_mass is unstable, which means it may not be tested or settled yet. Please avoid using this function in production code.

Parameters

  • markers (TimeSeries) – TimeSeries that contains marker trajectories as Nx4 during an action.

  • segments (Union[str, Sequence[str]]) – Name of the segment. Can be ‘Pelvis’, ‘Thorax’, ‘HeadNeck’, ‘ArmR’, ‘ArmL’, ‘ForearmR’, ‘ForearmL’, ‘HandR’, ‘HandL’, ‘ThighR’, ‘ThighL’, ‘LegR’, ‘LegL’, ‘FootR’, or ‘FootL’. An empty string or an empty list will try to process every segment.

  • sex (str) – Optional. Either ‘M’ or ‘F’. The default is ‘M’.

Returns

A TimeSeries with the trajectory of the segments’ centers of mass, named {segment}CenterOfMass.

Return type

TimeSeries

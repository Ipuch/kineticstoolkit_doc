kineticstoolkit.TimeSeries.ui_sync

kineticstoolkit.TimeSeries.ui_sync#

TimeSeries.ui_sync(data_keys=[], ts2=None, data_keys2=[])[source]#

Synchronize one or two TimeSeries by shifting their time.

If this method is called on only one TimeSeries, an interactive interface asks the user to click on the time to set to zero.

If another TimeSeries is given, an interactive interface allows synchronizing both TimeSeries together.

Warning

This function, which has been introduced in 0.1, is still experimental and may change signature or behaviour in the future.

Parameters

  • data_keys (Union[str, List[str]]) – Optional. The data keys to plot. If kept tempy, all data is plotted.

  • ts2 (Optional[TimeSeries]) – Optional. A second TimeSeries that contains both a recognizable zero-time event and a common event with the first TimeSeries.

  • data_keys2 (Union[str, List[str]]) – Optional. The data keys from the second TimeSeries to plot. If kept tempy, all data is plotted.

Returns

A copy of the TimeSeries after synchronization.

Return type

TimeSeries

Note

The TimeSeries ts2 is modified in place.

kineticstoolkit.pushrimkinetics.find_recovery_indices

find_recovery_indices(Mz, /)[source]

Find recovery indices based on a vector of propulsion moments.

Warning

The function find_recovery_indices is unstable, which means it may not be tested or settled yet. Please avoid using this function in production code.

This function analyzes the Mz moments to find which data correspond to pushes and which data correspond to recoveries. The method is very conservative on what could be considered as a recovery, so that every index returned by this function is almost certain to correspond to a recovery. This function is used by pushrimkinetics.remove_sinusoids to identify the instants with no hand contact. It should not be used to isolate the push and recovery phases (use ktk.cycles.detect_cycles() instead).

Parameters

Mz (ndarray) – Array that contains the propulsion moments in Nm.

Returns

Array of bools where each True represents recovery.

Return type

np.ndarray

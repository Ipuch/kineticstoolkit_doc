kineticstoolkit.TimeSeries.to_dataframe¶

TimeSeries.to_dataframe()¶

Create a DataFrame by reshaping all data to one bidimensional table.

Returns

DataFrame with the index as the TimeSeries’ time. Vector data are converted to single columns, and two-dimensional (or more) data are converted to multiple columns with the additional dimensions in brackets. The TimeSeries’s events and metadata such as time_info and data_info are not included in the resulting DataFrame.

Return type

DataFrame

kineticstoolkit.TimeSeriesEvent¶

class kineticstoolkit.TimeSeriesEvent(time=0.0, name='event')¶

Bases: object

Define an event in a timeseries.

This class is rarely used by itself, it is easier to use the TimeSeries` methods to deal with events.

Example

>>> event = ktk.TimeSeriesEvent(time=1.5, name='event_name')
>>> event
TimeSeriesEvent(time=1.5, name='event_name')
